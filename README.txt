5525 Final Project
Objective
Train traditional classification ML models on image data to classify chess pieces.

Data
Chess piece pictures from multiple sources on Kaggle:

https://www.kaggle.com/datasets/krithiik/chess-pieces/code

https://www.kaggle.com/datasets/niteshfre/chessman-image-dataset/code

Approach
Preprocess the image data and apply SVMs (linear, RBF, customized) to classify.
